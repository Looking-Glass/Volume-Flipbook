# Volume-Flipbook
A fully-contained 2d paint and animation tool for volume

# Controls
`Pallete: P`  
`Brush Size: 1, 2 or 3`  
`Switch Slice: [ or ]`  
`Timeline: T`  
`Switch Frame: Left Arrow/Right Arrow`  
`Play/Pause: Enter`  
`New Blank Frame: B`  
`Duplicate Current Frame: C`  
`Delete Current Frame: Delete`  
`Undo: Z`  
`Redo: Y`
